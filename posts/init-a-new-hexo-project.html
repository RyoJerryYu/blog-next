使用 hexo 搭建博客

最近使用 hexo 搭建了一个博客，并打算挂载在 github page 上。 对之前的那个博客进行替代，并将之前的文章逐渐搬移过来。

使用的这个主题功能还是比较完善的。

我们可以尝试一下代码块高亮：

def func_echo(s: str):
    print(s)
 
 
class HelloPrinter:
    printer: Callable[[str]]
 
    def __init__(self, printer: Callable[[str]]):
        self.printer = printer
    
    def call(self, s: str):
        self.printer(s)
 
 
p = HelloPrinter(func_echo)
p.call("hello world!")

来几句 mermaid 吧
